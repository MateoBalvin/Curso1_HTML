--POSICIONAMIENTO DE ELEMENTOS--

Padding

Crea un espacio alrededor de un elemento pero dentro del propio elemento, quiere decir que crea un espacio adentro mientras que el margin lo crea afuera

Margin

Crea un espacio entre los elementos de Html, es decir crea un espacio en blanco para un parrafo y otro, con otras palabras crea un espacio afuera del elemento mientras que el padding lo crea adentro
